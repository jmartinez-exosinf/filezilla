# ✅ Conexión a un servidor con FileZilla
Requisitos previos
Antes de comenzar, asegúrate de tener:

FileZilla Client instalado (Descargar aquí).
Datos de conexión del servidor:

Host: ejemplo ftp.tusitio.com
Usuario: ejemplo admin
Contraseña: la que te proporcionaron
Puerto: normalmente 21 para FTP o 22 para SFTP




1. Abrir FileZilla
Inicia la aplicación FileZilla Client.

2. Configurar la conexión rápida
En la barra superior verás estos campos:

Servidor (Host): escribe ftp.tusitio.com
Nombre de usuario: admin
Contraseña: tu clave
Puerto: 21 (o 22 si es SFTP)

Luego haz clic en Conexión rápida.

3. Verificar la conexión

Si todo está correcto, el panel derecho mostrará los archivos del servidor.
El panel izquierdo muestra tus archivos locales.


4. Transferir archivos

Para subir un archivo: arrástralo desde el panel izquierdo (tu PC) al derecho (servidor).
Para descargar: arrastra desde el panel derecho al izquierdo.


5. Usar el Gestor de Sitios (opcional)
Para guardar la conexión:

Ve a Archivo > Gestor de sitios.
Haz clic en Nuevo sitio.
Ingresa los mismos datos (Host, Usuario, Contraseña, Puerto).
Haz clic en Conectar y luego en Aceptar para guardar.


Consejos de seguridad

Si tu servidor soporta SFTP, úsalo en lugar de FTP (más seguro).
Evita usar contraseñas débiles.
Activa el modo pasivo si tienes problemas de conexión (en Opciones de transferencia).
